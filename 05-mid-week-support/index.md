## 5/21-22 온라인 교육

1주차 중 진행되는 미니 교육 주제입니다.

### AS-IS / TO-BE 정리법
- 현재 업무 흐름을 다이어그램으로 표현하는 방법
- AI Native 흐름으로의 전환점을 식별하는 관점

### 비민감 자료 만들기
- 개인정보·민감정보가 포함된 데이터를 익명화하는 방법
- 더미 데이터 생성 가이드

### 테스트 질문 작성법
- 하네스가 잘 동작하는지 확인하기 위한 질문 설계
- 정상 케이스 · 엣지 케이스 · 오류 처리

## 맞춤형 교재 요청 (MAX Studio)

본인 팀 과제에 맞춘 참고자료가 필요할 때 운영자에게 요청 가능합니다.

- **무엇이 가능한가**: 주제만 입력하면 MAX Studio가 PDF·PPTX·EPUB을 동시 생성
- **요청 방법**: 카카오 오픈채팅방 또는 운영자에게 카카오톡
- **예시**: "비스포크 가전 출점 후보지역 분석 방법론" 같은 도메인 자료
