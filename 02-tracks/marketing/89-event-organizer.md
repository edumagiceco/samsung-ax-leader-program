# 89. 이벤트 기획·운영 (event-organizer)
{: .fs-7 }

신규 오픈·특화 매장 이벤트를 컨셉부터 사후평가까지 풀패키지로
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신규 매장 그랜드오픈 / 반려동물·전기차 등 특화 매장 런칭 / 시즌 판촉
⏱️ **소요 시간** 90분 (풀실습)
🧰 **준비물** 매장 위치·평수·타깃 고객층, 예산 가이드
{: .fs-4 }

---

## 0. 에이전트 구성

```
🤖 이벤트기획자    → 컨셉·테마·메인 프로그램
🤖 로지스틱스매니저 → 장소·물품·일정 체크리스트
🤖 프로그램디자이너 → 시간대별 활동 설계
🤖 홍보담당        → SNS·전단·언론 보도자료
🤖 평가자          → 사후 KPI·다음 이벤트 개선안
```

---

## 1. 시나리오

> 강북구에 **반려동물 동반 가능 특화 매장**이 새로 오픈합니다.
> 오픈 3주 전, 그랜드오픈 이벤트를 기획해야 합니다.
> - 매장 평수: 150평
> - 예상 방문객: 첫 주말 500명
> - 예산: 1,500만 원
> - 주변 환경: 강북구 주거 밀집, 30~40대 반려인 多

## 2. 그대로 붙여넣기

### 🧩 단계 1 — 컨셉·테마

```
삼성전자판매 반려동물 동반 특화 매장 그랜드오픈 이벤트를 기획해줘.

매장 정보:
- 위치: 강북구 주거 밀집지역
- 평수: 150평
- 예산: 1,500만 원
- 일정: 3주 후 토·일 양일

다음을 제안해줘:
1) 이벤트 컨셉 키워드 3개
2) 메인 슬로건 (캐치프레이즈)
3) 메인 프로그램 3가지 (반려동물 가족 친화적)
4) 차별화 포인트 (일반 매장과 다른 점)
```

### 🧩 단계 2 — 로지스틱스 체크리스트

```
위 컨셉으로 이벤트 운영에 필요한 체크리스트를 만들어줘.

카테고리:
- 장소 세팅 (반려동물 안전·동선·위생)
- 물품 (가전 전시·이벤트 부스·반려동물 용품)
- 인력 배치 (영업·이벤트 진행·반려동물 케어)
- 외부 협력사 (반려동물 전문 트레이너·수의사·간식 업체)
- 일정 (D-21부터 D-Day까지 주차별)

표 형식, 담당자란 포함.
```

### 🧩 단계 3 — 시간대별 프로그램

```
이벤트 양일 시간대별 프로그램을 짜줘.

조건:
- 토·일 각 10:00-19:00
- 시간당 1개 이상 활동
- 반려동물 동반 가족 위주
- 가전 판매와 자연스럽게 연결되는 프로그램

각 프로그램:
- 시간·장소·진행자
- 소요 인력·물품
- 목표 (관심 유도·체험·구매전환)
```

### 🧩 단계 4 — 홍보 자료

```
이벤트 홍보 자료 세트를 만들어줘.

생성물:
1) 인스타그램 피드 게시물 3개 (각 200자 + 해시태그 10개)
2) 인스타 스토리 시리즈 (4컷, 카운트다운용)
3) 매장 전단 헤드라인·서브카피
4) 지역 커뮤니티 게시글 (네이버 카페·맘카페용)
5) 보도자료 1장 (보도자료 표준 형식)

톤: 따뜻하고 가족적이며, 반려동물에 대한 진심이 느껴지게.
```

### 🧩 단계 5 — 사후 평가 양식

```
이벤트 종료 후 작성할 평가 보고서 템플릿을 만들어줘.

지표:
- 방문객 수 (목표 vs 실적)
- 체험 활동 참여율
- 즉시 매출 / 견적 등록
- 후속 매출 추적 (1주·1개월)
- SNS 노출·언급 수
- 개선 액션 5가지

다음 이벤트에 그대로 적용할 수 있는 표준화 양식으로.
```

## 3. 결과 예시

### 컨셉
> **슬로건**: "온 가족이 함께 사는 집, 가전도 가족처럼"
> 키워드: #반려가족 #함께사는집 #케어가능한가전

### 프로그램 (일부)
| 시간 | 활동 | 가전 연결 |
|---|---|---|
| 11:00 | 반려동물 동반 입장 의식 + 기념사진 | 비스포크 디스플레이 앞 |
| 13:00 | 수의사 상담회 (15분 무료) | 공기청정기 체험존 |
| 15:00 | "반려동물 털 청소" 가전 시연 | 무선청소기 비교 체험 |

### 홍보 인스타 캡션 (예시)
> "강북점이 우리 가족, 진짜 모든 식구를 환영합니다 🐶🐱 11/15-16 그랜드오픈! 반려동물과 함께 매장 구경하세요. #반려가족 #삼성스토어강북점 ..."

## 4. 응용 팁

### 🎯 전기차 충전소 특화매장 버전
같은 흐름에 매장 컨셉만 바꾸면 됩니다.

```
이번엔 전기차 충전소 특화 매장 오픈. 매장 정보를 [전기차 차주 30~40대, 충전 대기 시간 활용] 으로 바꿔서 다시 기획해줘.
```

### 🎯 시즌 판촉 캠페인
- 봄: 신혼·이사철 → 패키지 이벤트
- 여름: 에어컨 집중 → 시연·할인
- 가을: 김장 → 김치냉장고 체험
- 겨울: 연말 보너스 → TV·고가 가전

### 🎯 평가 보고서를 본사에 제출
사후 평가 결과는 본사 마케팅 정례에 제출. 다음 매장 오픈 시 벤치마크.
