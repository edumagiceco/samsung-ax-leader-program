# 10. 매장 SNS 운영 (social-media-manager)
{: .fs-7 }

매장 인스타·블로그를 월간 콘텐츠 캘린더로 운영
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 매장 SNS 운영 / 시즌 콘텐츠 / 단골 소통
⏱️ **소요 시간** 15분
🧰 **준비물** 매장 인스타 컨셉, 지난달 인기 게시물 자료
{: .fs-4 }

## 그대로 붙여넣기

```
강남점 인스타그램 다음 달 콘텐츠 캘린더(30일치)를 만들어줘.

[매장 톤]
- 친근, 가족적, 라이프스타일
- 30~40대 주부·신혼 타깃

[혼합 비율]
- 신제품 소개 30%
- 라이프스타일 팁 30%
- 매장 이벤트·후기 20%
- 직원·매장 일상 20%

[월별 주요 일정]
- 둘째 주말: 비스포크 신컬러 출시
- 셋째 주: 어버이날
- 매주 토요일: 매장 체험 이벤트

각 게시물:
- 날짜·요일·시간
- 캡션 200자
- 해시태그 10개
- 이미지 가이드
```

## 응용 팁

- 인기 게시물 패턴을 분석해 다음 달에 반영
- 릴스 스크립트 (15초)도 같은 흐름으로 생성
- 댓글 응대 표준 멘트 세트 만들기
