# 03. 단골 고객 뉴스레터 (newsletter-engine)
{: .fs-7 }

매장 단골에게 보내는 월간 뉴스레터를 자동 생성
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 단골 고객 리텐션 / 신제품 안내 / 매장 이벤트 사전 공지
⏱️ **소요 시간** 15분
🧰 **준비물** 단골 고객 리스트(익명화), 이달의 매장 이슈
{: .fs-4 }

## 그대로 붙여넣기

```
강남점 단골 고객용 5월 뉴스레터를 만들어줘.

[이달의 매장 소식]
- 비스포크 신컬러 4종 출시
- 어버이날 효도 패키지 (10% 할인)
- 매장 체험존 리뉴얼 완료

[뉴스레터 구성]
1) 제목 (열고 싶은 제목)
2) 인사말 (3줄, 친근하게)
3) 신제품 소개 (제품별 100자)
4) 단골 전용 혜택 (3가지)
5) 이달의 매장 이벤트 안내
6) 점장 한마디 (5줄)
7) 발신 정보

분량: A4 1.5장 (메일 본문 기준)
톤: 친근한 단골 손님 안부 인사 느낌
```

## 응용 팁

- A/B 테스트용 제목 5개 같이 만들기
- 응답률 높은 콘텐츠 패턴 누적 추적
- 카톡 채널 알림톡 버전으로 변형

## 체크리스트

- [ ] 월 1회 정기 발송 일정 등록
- [ ] 오픈율·클릭률 추적
