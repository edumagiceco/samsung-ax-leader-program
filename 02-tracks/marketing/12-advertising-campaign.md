# 12. 시즌 판촉 캠페인 (advertising-campaign)
{: .fs-7 }

혼수·이사·연말 시즌별 캠페인 풀패키지
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 봄 신혼 / 여름 에어컨 / 가을 김장 / 겨울 연말 보너스
⏱️ **소요 시간** 15분
🧰 **준비물** 시즌 타깃, 본사 캠페인 정책, 매장 재고
{: .fs-4 }

## 그대로 붙여넣기

```
봄 혼수 시즌 (3~5월) 강남점 판촉 캠페인을 기획해줘.

[타깃]
- 30대 예비 신혼부부
- 평균 예산 700~900만 원
- 84㎡ 신혼집

[산출물]
1) 캠페인 메인 슬로건
2) 핵심 메시지 3개
3) 채널별 카피
   - 매장 전단 (헤드라인+서브)
   - 인스타 광고 (3가지 버전)
   - 네이버 검색광고 (3개 제목)
   - 매장 윈도우 디스플레이
4) 프로모션 구성
   - 패키지 할인 구조
   - 사은품 옵션
   - 무이자 할부
5) KPI (방문객·견적 등록·실매출)
```

## 응용 팁

- 본사 캠페인을 강남점에 맞춰 로컬라이즈
- 작년 같은 시즌 데이터와 비교
- 시즌 끝나면 결과 보고서 본사 제출
