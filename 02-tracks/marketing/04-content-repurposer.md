# 04. 콘텐츠 리퍼포징 (content-repurposer)
{: .fs-7 }

1개 제품 콘텐츠를 SNS·블로그·전단·POP로 다중 변환
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신제품 출시 / 본사 콘텐츠 매장 적용 / 한 제품을 여러 채널에
⏱️ **소요 시간** 15분
🧰 **준비물** 원본 콘텐츠 (본사 보도자료·제품 페이지 등)
{: .fs-4 }

## 그대로 붙여넣기

```
다음 비스포크 신형 냉장고 원본 보도자료를
매장에서 쓸 6가지 포맷으로 변환해줘.

[원본 붙여넣기]
삼성전자는 ... (본사 보도자료 전체 텍스트)

[변환할 포맷]
1) 인스타 캡션 (200자 + 해시태그)
2) 인스타 릴스 스크립트 (15초 영상 대본)
3) 매장 윈도우 POP 헤드라인 (한 줄)
4) 매장 전단 본문 (A4 1/4 분량)
5) 단톡방 안내 메시지 (100자)
6) 영업사원 응대 스크립트 (1분)

각 포맷은 매장 현장에 맞게 톤·길이 조정.
```

## 응용 팁

- 본사가 발행한 콘텐츠를 매장 톤으로 자동 변환
- 시간당 콘텐츠 생산성 5배 향상
- 매장별 로컬 색을 더해 차별화

## 체크리스트

- [ ] 신제품 출시 즉시 6포맷 발행 체계 구축
- [ ] 본사·매장 콘텐츠 일관성 유지
