# 13. 발표 자료 (presentation-designer)
{: .fs-7 }

매출 보고·신제품 브리핑·본사 발표 자료
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 본사 점장회의 / 신제품 브리핑 / 분기 평가 발표
⏱️ **소요 시간** 15분
🧰 **준비물** 발표 주제, 핵심 메시지, 데이터
{: .fs-4 }

## 그대로 붙여넣기

```
다음 주 본사 점장회의에서 발표할 자료 골격을 만들어줘.

[주제] 강남점 5월 실적 + 6월 액션 플랜
[시간] 15분 (질의 5분 별도)
[청중] 본사 영업본부, 타 매장 점장 20명

[구조]
1) 표지·제목
2) 핵심 메시지 한 장 요약
3) 5월 실적 하이라이트 (3장)
4) 잘된 점·아쉬운 점 (2장)
5) 6월 액션 3가지 (각 1장)
6) 본사 지원 요청 (1장)
7) 마무리·Q&A 유도

각 슬라이드:
- 제목
- 핵심 문구 (한 줄)
- 핵심 데이터·그래프 설명
- 발표 노트 (말로 할 내용)

총 12장 내외.
```

## 응용 팁

- 같은 데이터로 슬라이드·1장 요약·메모 보고 3종 동시 생성
- 발표 리허설 시 예상 질문 같이 만들기
- 본사 표준 템플릿 적용 가능

## 체크리스트

- [ ] 본사 발표 1주 전 슬라이드 1차 작성
- [ ] 점장·매니저 검토
