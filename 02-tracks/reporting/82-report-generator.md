# 82. 매출 보고서 (report-generator)
{: .fs-7 }

매장·카테고리·기간별 매출 보고서를 자동 생성
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 주간·월간 매출 보고 / 본사 정례 / 분기 평가
⏱️ **소요 시간** 15분
🧰 **준비물** 매출 데이터 (POS 추출, 익명화)
{: .fs-4 }

## 그대로 붙여넣기

```
강남점 5월 매출 보고서를 만들어줘.

[데이터]
- 총 매출: 12억 5천만 원 (전년 동월 대비 +8%)
- 카테고리: 냉장고 4억, TV 3.2억, 세탁기 2.8억, 에어컨 1.5억, 기타 1억
- 일별 매출 추이: ...
- 주요 이벤트: 어버이날 패키지

[보고서 구성]
1) 한 줄 요약
2) 핵심 지표 (전년 대비·목표 대비)
3) 카테고리 분석 (성장·정체·하락)
4) 일자별 추이 (그래프 가이드)
5) 이벤트 효과 분석
6) 다음 달 액션 3가지

분량: A4 3장
독자: 본사 영업본부
```

## 응용 팁

- 매주 같은 양식 → 누적 데이터로 트렌드 발견
- 매장별 비교 보고서로 확장
- 발표용 슬라이드 ([13 presentation-designer](../13-presentation-designer/))와 결합

## 체크리스트

- [ ] 월 1회 정례 보고 일정 등록
- [ ] 본사 표준 양식과 일치
