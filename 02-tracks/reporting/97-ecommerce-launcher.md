# 97. 온라인몰 상세페이지 (ecommerce-launcher)
{: .fs-7 }

samsungstore.com 상세페이지·프로모션 페이지 가속화
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신제품 온라인 등록 / 시즌 프로모션 페이지 / O2O 연계
⏱️ **소요 시간** 15분
🧰 **준비물** 제품 스펙, 핵심 차별점, 타깃 고객
{: .fs-4 }

## 그대로 붙여넣기

```
samsungstore.com에 비스포크 신컬러 출시 페이지를 만들어줘.

[제품] 비스포크 4도어 냉장고 신컬러 "코랄 핑크"
[타깃] 30~40대 여성, 컬러풀한 주방 추구

[페이지 구조]
1) 메인 비주얼 (히어로) — 한 줄 카피
2) 핵심 메시지 3개 (왜 사야 하나)
3) 디자인 스토리 (왜 이 컬러)
4) 스펙 표
5) 라이프스타일 사진 가이드 5컷
6) FAQ 5개
7) 매장 체험 안내 (O2O 연결)
8) 구매 CTA (할인·할부·매장 방문)

톤: 감성·디자인 중심, 라이프스타일 잡지 느낌
```

## 응용 팁

- 매장 인스타와 온라인몰 메시지 통일
- O2O: "온라인에서 보고 강남점에서 실물 확인" 동선 설계
- A/B 테스트용 카피 3개 생성

## 체크리스트

- [ ] 본사 e커머스팀과 컨텐츠 합의
- [ ] 매장 직원이 온라인 페이지 숙지
