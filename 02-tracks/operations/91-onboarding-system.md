# 91. 신입 직원 온보딩 (onboarding-system)
{: .fs-7 }

신규 매장 직원의 30-60-90일 체계화
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신입 직원 입사 / 신규 매장 오픈 / 점장 교체 / 부서 이동
⏱️ **소요 시간** 15분
🧰 **준비물** 직무 정의, 기존 베테랑 직원의 노하우 메모
{: .fs-4 }

## 그대로 붙여넣기

```
신입 매장 영업사원 90일 온보딩 계획을 만들어줘.

[직무]
- 가전·IT 제품 매장 영업·판매
- 고객 응대·견적·계약 클로징

[온보딩 단계]
1) Day 1-7: 매장 적응
2) Day 8-30: 제품·시스템 학습
3) Day 31-60: 멘토 동행 실전
4) Day 61-90: 독립 영업 + 평가

각 단계:
- 학습 목표 3가지
- 일일 활동
- 멘토 역할
- 평가 체크포인트
- 다음 단계로 가는 기준

부록: 90일 후 평가 시트
```

## 응용 팁

- 본사 인사팀 온보딩 표준과 매장 특화 부분 결합
- 멘토 매뉴얼 별도 작성
- 30일·60일·90일 차 본인 피드백 인터뷰 양식

## 체크리스트

- [ ] 신입 입사 즉시 90일 계획 전달
- [ ] 멘토 사전 교육·매칭
