# 84. 영업회의 운영 (meeting-strategist)
{: .fs-7 }

주간 영업회의·점장회의를 의사결정 중심으로
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 주간 매장 회의 / 본사 점장회의 / 분기 전략 미팅
⏱️ **소요 시간** 15분
🧰 **준비물** 회의 안건, 지난 회의 액션 아이템
{: .fs-4 }

## 그대로 붙여넣기

```
이번 주 강남점 주간 영업회의 아젠다와 진행안을 만들어줘.

[배경]
- 지난주 매출: 목표 대비 92%
- 이슈: 비스포크 신컬러 재고 부족
- 이벤트: 어버이날 패키지 진행 중
- 신입 사원 2명 첫 주

[회의 구조]
1) 오프닝 (5분) — 한 주 핵심 메시지
2) 지난주 리뷰 (15분) — 매출·이벤트·이슈
3) 이번주 액션 (20분) — 우선순위 3개
4) 신입 사원 발표 (10분) — 첫 주 소감
5) Q&A·이슈 토론 (10분)
6) 클로징 (5분)

산출물:
- 진행자 큐시트
- 슬라이드 5장 골격
- 회의록 템플릿
- 후속 액션 추적 양식
```

## 응용 팁

- 회의 끝나면 즉시 회의록 자동 생성
- 누적 액션 아이템 추적 대시보드
- 본사 보고 시 동일 양식 재활용

## 체크리스트

- [ ] 매주 같은 양식으로 회의 운영
- [ ] 액션 아이템 완료율 추적
