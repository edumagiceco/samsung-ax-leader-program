# 92. 매장 운영 매뉴얼 (operations-manual)
{: .fs-7 }

흩어진 문서를 모아 매장 운영 매뉴얼·플로차트·교육자료까지 자동 생성
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신규 매장 오픈 / 점장 교체 / 매뉴얼 노후화 / 본사 점검 대비
⏱️ **소요 시간** 90분 (풀실습)
🧰 **준비물** 기존 매장 문서 (개점·마감 체크리스트, 본사 가이드 발췌 등)
{: .fs-4 }

---

## 0. 에이전트 구성

```
🤖 문서분석가      → 기존 문서·메모 통합 분석
🤖 플로차트설계자  → 프로세스 시각화
🤖 매뉴얼작성자    → 단계별 설명서
🤖 FAQ빌더         → 매뉴얼 + 자주 묻는 질문
```

---

## 1. 시나리오

> 강남점 김 점장은 **3년간 누적된 메모·카톡·본사 공문**으로 매장이 운영되고 있다는 사실을 깨달았습니다.
> 신입 직원이 들어오면 매번 사람이 따라다니며 알려주는 비효율.
> 이번 분기 안에 **공식 매장 운영 매뉴얼 v1**을 만들기로 결정했습니다.

## 2. 그대로 붙여넣기

### 🧩 단계 1 — 기존 문서 통합

```
나는 가전 매장 점장이야.
다음 흩어진 문서·메모를 분석해서 매장 운영 영역을 분류해줘.

분류 기준:
- 영역 (개점·마감, 고객응대, 재고, 배송, 인사, 비상)
- 빈도 (매일·주간·월간·이벤트성)
- 책임자 (점장·매니저·직원)

[문서]
- 개점 체크리스트.txt (직원 8시 도착, 청소, 시스템 ON ...)
- 본사 공문 2025-Q4 (멤버십 정책 변경 ...)
- 카톡 메모: "월말 재고 실사 매월 말일"
- ... (붙여넣기)
```

### 🧩 단계 2 — 프로세스 플로차트

```
위에서 분류한 영역 중 다음 5개의 핵심 프로세스를
플로차트(텍스트 기반)로 그려줘.

1) 개점 절차 (8:00 - 10:00)
2) 마감 절차 (20:30 - 21:30)
3) 고객 응대 (입장 → 상담 → 견적 → 클로징)
4) 배송·설치 (주문 접수 → 일정 조율 → 사후 확인)
5) 비상 상황 (정전·시스템 장애·고객 컴플레인)

각 노드:
- 누가
- 무엇을
- 언제까지
- 다음 단계 조건
```

### 🧩 단계 3 — 단계별 매뉴얼

```
플로차트를 매뉴얼 문서로 풀어줘.

각 영역:
- 목적 (왜 이렇게 하나)
- 사전 준비물
- 단계별 절차 (번호 매김)
- 각 단계 소요 시간
- 자주 발생하는 실수와 방지법
- 문제 발생 시 연락처

분량: 영역당 A4 2장
톤: 신입 직원도 따라할 수 있게 친절하게.
```

### 🧩 단계 4 — FAQ 및 교육 자료

```
이 매뉴얼을 기반으로 신입 직원 첫 주 교육 자료를 만들어줘.

Day 1: 매장 둘러보기·기본 동선
Day 2: 개점·마감 셰도잉
Day 3: 고객 응대 시뮬레이션
Day 4: 시스템 사용·재고
Day 5: 실전 투입 + 멘토 옆에서

각 일자별:
- 학습 목표
- 시간표
- 평가 체크리스트
- 추가 학습 자료
```

## 3. 결과 예시

### 영역 분류
| 영역 | 빈도 | 책임자 |
|---|---|---|
| 개점 절차 | 매일 | 매니저 |
| 마감 절차 | 매일 | 점장 |
| 고객 응대 | 상시 | 전 직원 |
| 재고 실사 | 월 1회 | 매니저 |
| 배송 조율 | 주문별 | 영업사원 |

### 개점 절차 플로차트 (요약)
```
[08:00] 매니저 도착 → 시스템 점검
   ↓
[08:30] 직원 출근 → 청소·진열 확인
   ↓
[09:30] 일일 미팅 (15분) → 어제 매출·오늘 목표
   ↓
[10:00] 매장 오픈
```

### 매뉴얼 예시 (개점 절차)
> **목적**: 매일 10시 오픈 전 매장 상태를 100% 표준으로 맞추기.
>
> **사전 준비물**: 청소 도구, POS 시스템 ID, 전시 제품 상태 점검 체크리스트.
>
> **단계 1 (08:00, 매니저)**: 출입문 알람 해제 → 조명 ON → POS 시스템 부팅. 평균 10분.
> **자주 발생하는 실수**: 알람 해제를 놓치면 보안 출동. → 출근 즉시 알람 패널 먼저 확인.

## 4. 응용 팁

### 🎯 매장별 커스터마이즈
강남점 매뉴얼을 만든 뒤, 코엑스점·분당점에 맞게 변형. 80%는 공통, 20%만 다듬으면 됨.

### 🎯 본사 점검 대비
본사 점검 전 매뉴얼 vs 실제 운영 차이 점검:
```
이 매뉴얼대로 운영되고 있는지 자가 점검 체크리스트 50개 항목으로 만들어줘.
```

### 🎯 분기별 갱신
정책 변경·신제품 출시 시 매뉴얼 갱신. 변경 이력 관리도 같은 흐름으로.

## 5. 현장 적용 체크리스트

- [ ] 매뉴얼 v1 PDF로 제작 → 매장 내 비치
- [ ] 직원 단톡방에 핵심 플로차트 공유
- [ ] 신입 첫 주 교육 일정에 통합
- [ ] 분기 1회 갱신 일정 등록
- [ ] 본사에 매뉴얼 공유 → 타 매장 벤치마크 기회
