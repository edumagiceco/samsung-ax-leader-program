# 83. 표준운영절차 SOP (sop-writer)
{: .fs-7 }

개점·마감·재고·배송 SOP를 누구나 따라할 수 있게
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 매장 표준화 / 신입 교육 / 본사 점검 대비 / 매니저 교체
⏱️ **소요 시간** 15분
🧰 **준비물** 표준화할 업무 영역, 현재 운영 방식 메모
{: .fs-4 }

## 그대로 붙여넣기

```
매장 "월말 재고 실사" SOP를 만들어줘.

[현재 방식]
- 매월 말일 마감 후 직원 전원 참여
- 카테고리별 분담
- 3시간 소요
- 차이 발생 시 점장에게 보고

[SOP 구조]
1) 목적
2) 적용 범위·책임자
3) 사전 준비 (D-1)
4) 본 절차 (단계별 + 소요 시간)
5) 차이 발견 시 대응
6) 결과 기록·보고
7) 자주 발생하는 실수 5가지
8) 변경 이력 관리

분량: A4 2장
형식: 누구나 따라할 수 있는 체크리스트형
```

## 응용 팁

- 매장 운영 영역별 SOP 라이브러리 구축
- 동영상 SOP로 확장 (텍스트 → 영상 스크립트)
- 본사 표준 매뉴얼과 매장 SOP 차이 분석

## 체크리스트

- [ ] 영역별 SOP 5종 우선 작성 (개점·마감·재고·배송·응대)
- [ ] 분기 1회 갱신
