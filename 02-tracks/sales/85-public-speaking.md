# 85. 제품 시연 & 화법 (public-speaking)
{: .fs-7 }

신제품 시연·고객 응대 화법·예상 Q&A 한 번에
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신제품 출시 / 시연 행사 / 매장 응대 표준화 / 신입 교육
⏱️ **소요 시간** 15분
🧰 **준비물** 시연할 제품의 핵심 스펙·차별점
{: .fs-4 }

## 그대로 붙여넣기

```
삼성 비스포크 신형 냉장고를 매장에서 고객에게 시연해야 해.
다음을 만들어줘.

1) 1분 핵심 소개 멘트 (구어체)
2) 3분 풀시연 스크립트 (단계별 동작 + 멘트)
3) 예상 Q&A 7개와 모범 답변
4) "비교 모델보다 비싼데 왜 이게 좋아요?" 응대법
5) 클로징 멘트 3가지 버전 (구매 의향 강·중·약)

톤: 자신감 있되 부담스럽지 않게.
```

## 응용 팁

- 시연 영상 촬영 시 스크립트로 활용
- 신입 직원 응대 연습용 시뮬레이션 자료
- 본사 신제품 교육 시 매장 적용 버전 만들기
