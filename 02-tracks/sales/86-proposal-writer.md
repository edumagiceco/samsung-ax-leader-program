# 86. 법인 영업 제안서 (proposal-writer)
{: .fs-7 }

B2B·단체 구매 제안서를 빠르게 완성
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 법인 단체 구매 / 사옥 일괄 납품 / 호텔·기숙사 대량 수주
⏱️ **소요 시간** 15분
🧰 **준비물** 고객사 정보, 납품 규모·일정, 예산 가이드
{: .fs-4 }

## 그대로 붙여넣기

```
삼성전자판매 법인영업 담당자야. 다음 고객사용 제안서 초안을 만들어줘.

[고객사 정보]
- 회사: A호텔 (가명, 200객실)
- 요구: 객실용 TV 200대, 미니냉장고 200대 일괄 납품
- 일정: 3개월 후 리뉴얼 오픈
- 예산: 약 8억 원
- 우려: 설치 일정·AS 응답 속도

[제안서 구성]
1) 표지·요약 (1장)
2) 우리의 강점 (전국 매장망·AS)
3) 추천 제품 구성 + 단가표
4) 일정 계획 (3개월 분할 납품)
5) AS·사후관리 SLA
6) 가격 (정가 vs 제안가, 절감액)
7) 결제 조건
```

## 응용 팁

- RFP 응답서로 확장 ([55 rfp-responder](https://github.com/revfactory/harness-100/tree/main/ko/55-rfp-responder))
- 매장 점장이 지역 법인 영업 시 활용
- 제출 전 매니저·점장 검토 필수 (가격 권한 확인)

## 체크리스트

- [ ] 본사 법인영업 가이드와 가격 정책 일치 확인
- [ ] 첨부 자료 (회사 소개·매장망) 표준화
