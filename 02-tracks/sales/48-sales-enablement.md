# 48. 영업 인에이블먼트 (sales-enablement)
{: .fs-7 }

매장 방문 고객의 니즈를 분석해 **맞춤 견적·제안서·팔로업**까지 한 번에
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 신혼부부 패키지 상담 / 이사철 가전 일괄 견적 / 법인 단체 구매 문의
⏱️ **소요 시간** 90분 (풀실습)
🧰 **준비물** 고객 메모(가공·익명화), 견적 대상 제품 카테고리 리스트
{: .fs-4 }

---

## 0. 들어가기 전

이 사례는 **4개의 AI 직원(에이전트)**을 팀으로 묶습니다.

```
🤖 고객분석가      → 페르소나·니즈·예산 정리
🤖 제안서작성자    → 제품 조합 + 견적서 초안
🤖 프레젠터        → 매장 응대 스크립트
🤖 팔로업매니저    → 사후 연락 메시지
        ↑
   🎯 오케스트레이터가 흐름 지휘
```

{: .warning }
> **개인정보 입력 금지** — 실제 고객 실명·연락처·계좌는 절대 AI에 입력하지 마세요. 본 실습에서는 "신혼부부 김OO 고객(가명)" 같이 익명화된 페르소나만 사용합니다.

---

## 1. 시나리오 (실제 매장 상황)

> 평일 오후 2시. 30대 신혼부부가 매장에 방문해 **혼수 가전 패키지** 견적을 문의했습니다.
> - 예산: 약 800만 원
> - 거주: 84㎡ 아파트, 입주 2주 후
> - 우선순위: 디자인 통일감, AS 편의성
> - 우려: 가격, 설치 일정

## 2. 그대로 붙여넣기 (프롬프트 카드)

터미널에서 `claude` 실행 후 다음을 복사·붙여넣기.

### 🧩 단계 1 — 고객 분석가 호출

```
나는 가전 매장 영업사원이야. 다음 고객 정보를 보고
1) 페르소나 요약 2) 숨은 니즈 3가지 3) 결정 기준 3가지
를 한국어로 정리해줘.

[고객 메모]
- 30대 신혼부부 (가명: 김OO·이OO)
- 예산: 약 800만 원
- 거주: 84㎡ 신축 아파트, 2주 후 입주
- 우선순위: 디자인 통일감, AS 편의성
- 우려: 가격, 설치 일정 (입주 전까지 모든 설치 희망)
- 관심 제품: 냉장고, 세탁기/건조기, TV, 에어컨
```

### 🧩 단계 2 — 제안서 작성자

```
위 페르소나를 기반으로 800만 원 예산 안에서
삼성 가전 패키지 견적서 초안을 만들어줘.

요구사항:
- 카테고리별 제품 1순위/2순위 추천
- 각 제품의 추천 이유 1줄
- 패키지 할인 적용 가정 (10% 할인)
- 설치 일정 제안 (2주 내)
- 표 형식으로 정리
```

### 🧩 단계 3 — 매장 응대 스크립트

```
이 패키지를 매장에서 고객에게 설명할 때 쓸 스크립트를 만들어줘.

구조:
1) 첫 1분 — 핵심 가치 제안
2) 제품별 설명 포인트 (각 30초)
3) 예상 질문 5개와 답변
4) 마무리 클로징 멘트
구어체로, 매장에서 바로 말할 수 있는 톤으로.
```

### 🧩 단계 4 — 팔로업 메시지

```
오늘 상담 후 고객에게 보낼 카톡 메시지 3가지 버전을 만들어줘.
- 버전 A: 당일 저녁 (감사 + 견적서 전달)
- 버전 B: 3일 후 (결정 도움 + 추가 정보)
- 버전 C: 1주일 후 (마지막 리마인드, 부담 없게)
각 200자 이내, 존댓말.
```

## 3. 결과 예시

### 단계 1 결과 (요약)
```
페르소나: "디자인 일관성을 중시하는 실용형 신혼 커플"
숨은 니즈: ① 입주 일정 압박 ② 의사결정 단순화 ③ 사후관리 안심
결정 기준: ① 통일된 디자인 ② AS 편의 ③ 설치 일정 준수
```

### 단계 2 결과 (견적서)
| 카테고리 | 1순위 추천 | 추천 이유 |
|---|---|---|
| 냉장고 | 비스포크 4도어 (875L) | 디자인 패널 통일 |
| 세탁기 | 비스포크 그랑데 AI | 건조기 일체 패키지 |
| TV | 네오 QLED 65인치 | 거실 메인 디스플레이 |
| 에어컨 | 비스포크 무풍갤러리 | 통일 디자인 |
| **소계** | | 약 870만 원 → **10% 패키지 할인 783만 원** |

### 단계 3 결과 (스크립트 일부)
> "두 분 안녕하세요. 디자인 통일감이 가장 중요하다고 하셔서 비스포크 컬렉션 중심으로 준비했어요. 패널 색상을 모두 같은 톤으로 맞추면 입주하실 때 전체 가전이 하나의 컬렉션처럼 보입니다..."

## 4. 응용 팁

### 🎯 다른 고객 유형으로 확장
- "1인 가구 30대 남성, 예산 200만 원"
- "60대 부부, 노부모 부양, AS 편의 최우선"
- "법인 사내 휴게실, 직원 50명 사용"

각 페르소나를 단계 1에 입력하면 전체 흐름이 자동으로 맞춰집니다.

### 🎯 견적 협상 시뮬레이션
단계 2 결과를 가지고 다음 프롬프트로 협상 연습:

```
방금 견적에서 고객이 "100만원 더 깎아달라"고 했어.
원가 손실 없이 어떻게 응대할지 3가지 방안 제시해줘.
(예: 사은품 추가, 무이자 할부, 가성비 모델 대체 등)
```

### 🎯 스킬로 저장해 재사용
이 4단계를 매번 입력하는 게 번거롭다면 **스킬**로 저장. Day 2 프로젝트 키트에서 다룹니다.

## 5. 현장 적용 체크리스트

매장 복귀 후 다음을 점검하세요.

- [ ] 매장 상담 시 사용할 **표준 페르소나 3종** 만들기 (신혼·이사·법인)
- [ ] 견적서 출력 시 **하단에 본인 연락처 자동 포함** 되도록 설정
- [ ] 응대 스크립트를 **소형 카드**로 출력해 매장 비치
- [ ] 팔로업 메시지는 **카톡 매크로**에 저장
