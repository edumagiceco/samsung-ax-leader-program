# 50. 가격 전략 (pricing-strategy)
{: .fs-7 }

할인·번들·렌탈을 시뮬레이션해 최적 견적 조합 찾기
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 고객 협상 / 시즌 프로모션 / 본사 캠페인 적용 시 시뮬레이션
⏱️ **소요 시간** 15분
🧰 **준비물** 견적 대상 제품 정가, 본사 할인 정책
{: .fs-4 }

## 그대로 붙여넣기

```
가전 매장 영업사원이야. 다음 조건으로 견적 옵션 3가지를 시뮬레이션해줘.

[조건]
- 제품: 비스포크 냉장고(450만), 세탁기/건조기(280만), TV 65인치(200만)
- 본사 패키지 할인: 10%
- 멤버십 할인: 추가 3%
- 24개월 무이자 할부 가능
- 렌탈 옵션도 비교

[출력]
옵션 A: 일시불 + 최대 할인
옵션 B: 무이자 할부 (월 부담액)
옵션 C: 렌탈 vs 구매 5년 총비용 비교
각 옵션의 장단점 한 줄씩.
```

## 응용 팁

- "예산 800만으로 맞춰줘" 같은 역방향 견적도 가능
- 사은품 추가 시 효과 시뮬레이션
- 경쟁사 가격 대비 우위 포인트 추출

## 체크리스트

- [ ] 자주 쓰는 견적 조합 5개를 표준 프롬프트로 저장
- [ ] 매장 컴퓨터에 즐겨찾기
