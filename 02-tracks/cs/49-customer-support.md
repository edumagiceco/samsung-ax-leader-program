# 49. 고객지원 시스템 (customer-support)
{: .fs-7 }

매장 FAQ → 응대 매뉴얼 → 에스컬레이션 체계까지 한 번에
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 매장 단골 문의 정리 / 신입 직원 응대 교육 / AS 안내 표준화
⏱️ **소요 시간** 90분 (풀실습)
🧰 **준비물** 최근 3개월 매장 문의 메모 또는 카톡 캡처 (익명화)
{: .fs-4 }

---

## 0. 들어가기 전

이 사례는 **4개의 에이전트** 팀을 구성합니다.

```
🤖 FAQ빌더         → 자주 묻는 질문 분류·정리
🤖 응대전문가      → 질문별 응대 스크립트
🤖 에스컬레이션매니저 → 단계별 대응 흐름
🤖 CS분석가        → 패턴·트렌드 인사이트
```

{: .warning }
> **고객 실명·연락처는 반드시 익명화하세요.** "강남점 38번 고객" → "고객 A" 식으로.

---

## 1. 시나리오

> 강남점 김 매니저는 최근 3개월간 받은 매장 문의 80건을 정리하려고 합니다.
> 신입 직원이 들어와도 같은 품질로 응대할 수 있는 **FAQ + 응대 매뉴얼 + 에스컬레이션 흐름**을 만들고 싶습니다.

## 2. 그대로 붙여넣기

### 🧩 단계 1 — FAQ 분류

```
나는 삼성전자판매 매장 CS 담당자야.
최근 3개월간 받은 고객 문의 80건을 카테고리로 분류해줘.

분류 기준:
- 빈도 (자주 묻는 순)
- 처리 난이도 (즉답 / 조회 필요 / 본사 문의)
- 응대 시간 (5분 / 15분 / 30분 이상)

[문의 메모]
1. 비스포크 냉장고 패널 색상 변경 가능한가요?
2. 세탁기 배송 일정 변경하고 싶어요
3. AS 접수 후 며칠 걸리나요?
4. 멤버십 포인트로 결제 가능한가요?
... (남은 76건 붙여넣기 또는 요약)
```

### 🧩 단계 2 — 응대 스크립트

```
위에서 분류한 카테고리 중 빈도 Top 10에 대해
매장 응대 스크립트를 만들어줘.

각 스크립트 구조:
- 첫 응답 (10초): 고객 안심
- 정보 확인 (30초): 무엇을 확인할지
- 해결 제시 (1분): 옵션 1·2·3
- 마무리 (15초): 다음 단계 안내

톤: 정중하고 따뜻하게, 매장 응대 구어체
```

### 🧩 단계 3 — 에스컬레이션 흐름

```
즉답이 어려운 문의를 본사·전문가에게 넘기는
에스컬레이션 흐름도를 만들어줘.

레벨 분류:
- L1: 매장 직원 (즉답)
- L2: 매장 매니저 (정책·환불)
- L3: 본사 CS (AS 진행·하자)
- L4: 법무·소비자보호 (분쟁)

각 레벨의:
- 처리 범위
- 다음 레벨로 넘기는 조건
- 인계 시 필수 기재 사항
```

### 🧩 단계 4 — 인사이트 보고

```
이번에 정리한 80건의 문의에서 다음을 분석해줘.

1) 매장 운영 개선이 필요한 영역 Top 3
2) 본사에 정책 변경을 요청할 항목
3) 직원 교육이 부족한 분야
4) 다음 분기 매장 KPI에 반영할 지표 2개
```

## 3. 결과 예시

### FAQ 분류 결과 (요약)
| 순위 | 카테고리 | 빈도 | 난이도 |
|---|---|---|---|
| 1 | 배송·설치 일정 | 18건 | 즉답 |
| 2 | AS 접수·기간 | 14건 | 조회 |
| 3 | 결제·할부·멤버십 | 11건 | 즉답 |
| 4 | 제품 비교·추천 | 9건 | 즉답 |
| 5 | 환불·교환 | 8건 | 본사 |

### 응대 스크립트 예시 (배송 일정)
> "고객님, 배송 일정 변경 도와드리겠습니다. 먼저 주문번호 확인 부탁드릴게요. 시스템에서 가능한 일정 3개 안내드리고, 가장 편하신 시간으로 잡아드리겠습니다. 잠시만요..."

### 에스컬레이션 예시
```
[배송 지연 컴플레인]
L1 매장 직원: 시스템 조회 → 가능 일정 제시
   ↓ 고객 거부 / 일정 불가
L2 매장 매니저: 손해 보상 (사은품·할인) 권한
   ↓ 보상 거부 / 환불 요구
L3 본사 CS: 환불 절차 진행
   ↓ 법적 분쟁 언급
L4 법무팀
```

## 4. 응용 팁

### 🎯 신입 교육 자료로 변환
```
방금 만든 매뉴얼을 신입 직원 첫 주 교육 자료로 다시 정리해줘.
- 1일차: FAQ Top 5만
- 2일차: 응대 스크립트 연습
- 3일차: 에스컬레이션 시뮬레이션
```

### 🎯 매장 비치용 카드
응대 스크립트를 **A6 사이즈 카드**로 인쇄해 매장 카운터에 배치.

### 🎯 월간 업데이트
매달 새 문의를 추가해 FAQ를 갱신. 같은 프롬프트로 누적 분석 가능.

## 5. 현장 적용 체크리스트

- [ ] FAQ Top 10을 **매장 카운터 카드**로 출력
- [ ] 응대 스크립트를 **신입 교육 매뉴얼**에 통합
- [ ] 에스컬레이션 흐름을 **매장 게시판**에 부착
- [ ] 월간 인사이트 보고를 **본사 정례회의**에 제출
