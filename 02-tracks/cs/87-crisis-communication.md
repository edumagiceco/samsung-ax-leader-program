# 87. 위기 소통 (crisis-communication)
{: .fs-7 }

컴플레인·언론·리콜 등 위기 상황에서 침착한 메시지
{: .fs-5 .fw-300 }

---

🎯 **언제 쓰나요?** 강성 컴플레인 / 매장 사고 / 본사 리콜 공지 매장 대응 / 언론 문의
⏱️ **소요 시간** 15분
🧰 **준비물** 상황 요약, 사실관계, 대상자
{: .fs-4 }

## 그대로 붙여넣기

```
매장에서 위기 상황이 발생했어. 메시지 전략을 만들어줘.

[상황]
- 고객이 구매한 세탁기가 설치 다음 날 누수
- 카펫·바닥재 손상 추정 200만 원
- 고객은 매장에 전화로 강하게 항의 중
- SNS에 사진을 올리겠다고 언급

[필요 산출물]
1) 사과 + 즉시 대응 약속 첫 응대 멘트 (전화)
2) 현장 출동 일정 안내 (24시간 내)
3) 손해 보상 절차 안내문 (서면 발송용)
4) 만약 SNS에 올라간다면 댓글 응대 (3가지 강도)
5) 본사 보고 양식 (상황·대응·예상 리스크)

톤: 절대 변명하지 않음, 사실 인정, 신속한 행동 약속, 공감 우선
```

## 응용 팁

- 자주 발생하는 위기 유형 5개 표준 매뉴얼화
- 매장 점장은 본사 PR팀 직통 연락처 확보
- 절대 약속할 수 없는 것은 명시 (보상 한도 등)

## 체크리스트

- [ ] 본사 PR팀·법무팀 비상 연락망 매장 비치
- [ ] 위기 단계별 의사결정 권한 명확화
