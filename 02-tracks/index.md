## 페이지 읽는 법

각 사례 페이지는 다음 표준 구조를 따릅니다.

```
🎯 언제 쓰나요?      현장 시나리오
⏱️  소요 시간         예상 분 단위
🧰 준비물             데이터·문서 목록

1) 시나리오
2) 그대로 붙여넣기 (프롬프트 카드)
3) 결과 예시
4) 응용 팁
5) 체크리스트
```

⭐ 표시된 풀실습 사례는 60~90분, 나머지는 10~15분 분량입니다.
