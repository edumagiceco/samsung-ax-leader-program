# 심사 루브릭
{: .fs-7 }

심사위원이 사용하는 평가표. 참가자도 미리 알면 더 잘 준비할 수 있습니다.
{: .fs-5 .fw-300 }

---

## 평가 항목 4종 (총 100점)

### 1) 실용성 (40점) — "내일부터 쓸 수 있는가"

| 점수 | 기준 |
|---|---|
| 36-40 | 매장 적용 즉시 가능, 30일 챌린지 계획 구체적 |
| 28-35 | 일부 보완 후 적용 가능, 적용 의지 명확 |
| 20-27 | 컨셉은 좋으나 매장 환경에 추가 작업 필요 |
| 10-19 | 매장에 적용하려면 큰 변경 필요 |
| 0-9 | 매장 업무와 연관성 약함 |

### 2) 완성도 (30점) — "정말 작동하는가"

| 점수 | 기준 |
|---|---|
| 27-30 | 데모 완벽 실행, 다양한 입력 처리 |
| 21-26 | 데모 정상 작동, 기본 시나리오 OK |
| 14-20 | 데모는 보여주나 일부 막힘 |
| 7-13 | 부분만 작동, 시연 어려움 |
| 0-6 | 동작 안 함 |

### 3) 창의성 (20점) — "새로운 시각"

| 점수 | 기준 |
|---|---|
| 18-20 | 사례 조합·응용이 독창적, 다른 팀 모방 어려움 |
| 14-17 | 기존 사례를 매장 맥락에 잘 적용 |
| 9-13 | 워크샵 사례를 단순 활용 |
| 4-8 | 사례 그대로 따라함 |
| 0-3 | 워크샵 사례와 무관 |

### 4) 발표력 (10점) — "5분 안에 전달"

| 점수 | 기준 |
|---|---|
| 9-10 | 시간·구조·임팩트 완벽 |
| 7-8 | 메시지 명확, 시간 준수 |
| 5-6 | 전달은 되나 늘어짐 |
| 3-4 | 핵심 메시지 불명확 |
| 0-2 | 시간 초과·중단 |
