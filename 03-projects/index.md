# Day 2 프로젝트 키트
{: .fs-8 }

Day 1에서 배운 사례를 조합해 **우리 매장만의 AI 도우미 1개**를 만듭니다.
{: .fs-5 .fw-300 }

---

## Day 2 흐름

```
09:30  프로젝트 캔버스 작성       (45분)
10:15  본격 개발 1차              (120분)
12:15  점심
13:15  본격 개발 2차              (90분)
14:45  발표 자료 마무리           (45분)
15:30  팀별 5분 발표 + Q&A       (트랙당 3~4팀)
17:00  시상 + 30일 챌린지 안내
```

## 키트 구성

| 도구 | 용도 |
|---|---|
| [프로젝트 캔버스](./canvas/) | 문제→대상→산출물→평가까지 1장 정리 |
| [주제 풀](./topics/) | 트랙별 추천 프로젝트 주제 25개 |
| [발표 템플릿](./presentation-template/) | 5분 발표용 슬라이드 골격 |
| [심사 루브릭](./rubric/) | 실용성·창의성·완성도 평가표 |

## 조 편성 원칙

- **3~5명 1조** (5개 트랙 × 평균 4팀 = 20팀 내외)
- **같은 트랙 동료**끼리 묶기 (같은 직무 문제의식 공유)
- 각 조에 **멘토 1명** 배치 (운영진 또는 외부 코치)

## 발표 평가 기준

| 항목 | 배점 | 설명 |
|---|---|---|
| 실용성 | 40 | 우리 매장에서 내일부터 쓸 수 있는가 |
| 완성도 | 30 | 실제로 동작하는가, 결과물이 나오는가 |
| 창의성 | 20 | 시나리오의 신선함, 응용력 |
| 발표력 | 10 | 5분 안에 핵심 전달 |

상위 3팀에 시상하며, 우수작은 사내 채널에 공유됩니다.
