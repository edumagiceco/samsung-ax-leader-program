# 동작 확인 체크리스트
{: .fs-7 }

5분 안에 5가지 테스트를 끝내세요. 모두 통과해야 워크샵 준비 완료입니다.
{: .fs-5 .fw-300 }

---

## ✅ 체크 1 — 버전 확인

```
claude --version
```

**기대 결과**: `claude-code/x.x.x ...` 형식의 버전 표시
**실패 시**: [설치 가이드](../install-windows/) 다시 진행

## ✅ 체크 2 — 로그인 상태

```
claude /status
```

또는 `claude` 실행 후 `/status` 입력.

**기대 결과**: 본인 이메일 표시
**실패 시**: `claude /login` 으로 재로그인

## ✅ 체크 3 — 첫 대화

```
claude
```

실행 후 `>` 프롬프트에 다음을 입력:

```
안녕, 가전 매장에서 일하는 영업사원이야.
"30대 신혼부부에게 어떤 냉장고를 추천할지" 1줄로 답해줘.
```

**기대 결과**: 5초 안에 1~2줄 추천 답변 출력
**실패 시**: 인터넷 연결 확인 → [트러블슈팅](../troubleshooting/)

## ✅ 체크 4 — 파일 만들기 권한

```
간단한 메모 파일 hello.txt를 만들어줘. 내용은 "Hello workshop"
```

Claude Code가 파일을 만들기 전에 권한을 요청합니다. **허용** 선택.

```
ls hello.txt
```

파일이 생성되어 있으면 성공.

**기대 결과**: `hello.txt` 표시
**실패 시**: 폴더 권한 확인 (workshop 폴더가 본인 계정 소유인지)

## ✅ 체크 5 — 인증샷 제출

위 1~4번 화면을 캡처해서 **워크샵 단톡방**에 올리세요.

캡처 내용:
- `claude --version` 결과
- `claude` 실행 화면 (`>` 프롬프트가 보이는 상태)
- `hello.txt` 생성 결과

{: .important }
> **제출 마감: 워크샵 3일 전 (D-3)**
> 미제출자는 워크샵 당일 헬프데스크 줄을 서야 합니다.
