# Claude 계정 발급
{: .fs-7 }

소요 시간: 약 5분
{: .fs-5 .fw-300 }

---

## 1단계 — 가입 페이지 접속

[https://claude.ai/signup](https://claude.ai/signup) 접속

## 2단계 — 이메일 입력

회사 메일 또는 본인 메일을 입력하세요.

{: .tip }
> 워크샵에서 사용하려면 **개인 이메일**을 권장합니다. 회사 메일은 보안 정책상 외부 인증 메일이 차단되는 경우가 있습니다.

## 3단계 — 인증 메일 확인

받은 메일함에서 6자리 인증번호를 입력합니다.

- **메일이 안 오면?** 스팸 함 확인
- **회사 메일 차단?** 개인 메일로 다시 시도

## 4단계 — 프로필 입력

- 이름: 본인 실명 (워크샵 출석 확인용)
- 직책: "삼성전자판매 - [지점명]"
- 사용 목적: "업무 효율화"

## 5단계 — 요금제 선택

워크샵에서는 **무료 플랜**도 가능하지만, Day 2 프로젝트 발표를 위해서는 **Pro 플랜 (월 $20)** 또는 회사 제공 라이선스를 권장합니다.

| 플랜 | 한도 | 워크샵 적합도 |
|---|---|---|
| Free | 일 사용량 제한 | Day 1 입문 실습만 |
| Pro | 충분 | Day 1·2 모두 가능 ⭐ |
| Team | 충분 + 공유 | 사내 도입 시 |

{: .important }
> **회사가 라이선스를 제공하는지 IT팀에 확인하세요.** 사내 단체 라이선스가 있으면 비용 없이 Pro 기능을 쓸 수 있습니다.

## 6단계 — 로그인 확인

[https://claude.ai](https://claude.ai)에 다시 로그인하여 정상 접속되는지 확인합니다.
