# 시작 전 준비 (D-7)
{: .fs-8 }

워크샵 일주일 전, 본인 노트북에서 **반드시 사전 설치를 완료**하세요.
{: .fs-5 .fw-300 }

---

## 왜 미리 준비해야 하나요?

매장 PC와 회사 노트북은 보안 정책상 설치 권한, 방화벽 제한이 있을 수 있습니다. **워크샵 당일 설치를 시작하면 늦습니다.** 일주일 전 준비를 마치고, 동작 확인까지 완료해야 Day 1 첫 시간부터 실습에 참여할 수 있습니다.

## 준비 체크리스트 (총 약 30~40분)

| 단계 | 소요 | 페이지 |
|---|---|---|
| 1️⃣ Claude 계정 발급 | 5분 | [계정 발급하기](./account-setup/) |
| 2️⃣ Claude Code 설치 | 10~15분 | [Windows](./install-windows/) · [Mac](./install-mac/) |
| 3️⃣ 동작 확인 | 5분 | [체크리스트](./checklist/) |
| 4️⃣ 문제 발생 시 | — | [트러블슈팅](./troubleshooting/) |

## D-7 / D-3 / D-1 일정

```
D-7  (월)  계정 발급 + 설치
D-5  (수)  IT팀 지원 데스크 운영
D-3  (금)  "설치 완료 인증샷" 단톡방 제출
D-1  (월)  최종 점검 안내 메일
 D   (화)  워크샵 시작
```

{: .important }
> **인증샷 제출 마감: 워크샵 3일 전**
> 미제출자는 워크샵 당일 헬프데스크 줄을 서야 합니다.

## 필요한 것

- [ ] 본인 노트북 (Windows 10/11 또는 macOS 12+)
- [ ] 관리자 권한 (또는 IT팀 사전 협의)
- [ ] 안정적인 인터넷 (사내 Wi-Fi 또는 핸드폰 테더링)
- [ ] 회사 메일 또는 본인 메일 주소

## 매장 PC를 쓰시나요?

매장 비치 PC는 보안 제약이 큽니다. 다음 중 하나를 선택하세요.

1. **개인 노트북 지참** (권장)
2. **본사 IT팀과 사전 협의** 후 매장 PC에 설치 권한 받기
3. **백업 옵션**: 설치가 안 되면 [claude.ai](https://claude.ai) 웹 버전으로 부분 참여
