## 1주차 자료 준비 시 참고할 항목 (선택)

5/20-26 사이에 다음 항목들이 정리되어 있으면 5/27 당일 작업이 수월합니다. 강제 사항이 아닌 일반 가이드입니다.

| 영역 | 1주차 동안 정리해두면 좋은 것 |
|---|---|
| **과제 정의** | 어떤 업무 영역을 다룰지 (1-2개로 좁히기) |
| **AS-IS** | 현재 업무 흐름의 단계·소요 시간·자주 발생하는 문제 |
| **데이터** | 사용할 수 있는 비민감 자료 또는 더미 데이터 샘플 |
| **테스트 질문** | 하네스에 던질 테스트 질문 3-5개 (정상 케이스·엣지 케이스 포함) |
| **하네스 구성 안** | 도메인 전문가 4명 + reviewer 1명 + skills 3개의 1차 안 |
| **검증 기준** | 무엇으로 "잘 동작한다"고 판단할지 |

## 평가 기준 (참고)

5/27 발표는 다음 3가지로 평가됩니다.

- **효과성**: 실제 적용 시 생산성 향상이나 문제 해결에 효과가 기대된다
- **완성도**: 과제 정의부터 발표까지 완성도 있게 수행되었다
- **혁신성**: 과제 주제가 차별화되고 참신한 시도를 담고 있다
