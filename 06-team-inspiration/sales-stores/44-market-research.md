# 44. 시장 조사 (market-research)
{: .fs-7 }

산업·경쟁·소비자·트렌드를 통합 조사해 시장 환경 보고서를 작성하는 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 산업 구조·경쟁 구도·소비자 행동·트렌드 분석
- **출력 형태**: 시장 조사 보고서, 기회·위협 매핑
- **사용 빈도**: 신규 사업 검토·연간 전략 수립 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 산업 분석가 | 산업 규모·성장률·구조 분석 |
| 경쟁 분석가 | 경쟁사 포지셔닝·점유율·전략 분석 |
| 소비자 분석가 | 세그먼트·페르소나·구매 동인 |
| 트렌드 분석가 | 거시 트렌드·기술·규제 변화 |
| reviewer | 출처 신뢰도·인용 정확성 검증 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 4 영역 통합 보고서 구성 |
| 포지셔닝 매핑 방법론 | 2축 매트릭스 설계 |
| 트렌드 신호 방법론 | weak signal·STEEP 분석 |

## 비민감 데이터 활용 시 고려할 점

- 자체 매출·점유율은 비율 형태로 가공
- 외부 시장 자료는 공개 출처·인용 표기
- 미공개 사업 계획·미발표 신제품은 입력 금지

## 검증 시 일반적으로 보는 항목

- 인용 자료의 출처와 시점이 명시되어 있는가
- 4 영역 간 모순되는 결론이 없는가
- 도출된 기회·위협이 실행 가능한 수준인가
