# 12. 광고 캠페인 (advertising-campaign)
{: .fs-7 }

타깃 분석·카피·크리에이티브·미디어 플랜을 통합 설계하는 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 타깃 정의, 카피·크리에이티브, 채널 믹스, 미디어 플랜
- **출력 형태**: 캠페인 기획서, 채널별 소재, 예산·KPI 표
- **사용 빈도**: 시즌별·신제품 출시 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 마케터 | 타깃·핵심 메시지·캠페인 컨셉 정의 |
| 카피라이터 | 채널별 카피·헤드라인 작성 |
| 크리에이티브 디렉터 | 비주얼·톤앤매너 설계 |
| 미디어 플래너 | 채널 믹스·예산 배분·KPI 설정 |
| reviewer | 브랜드 가이드 정합성 검증 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 타깃→카피→크리에이티브→미디어 흐름 |
| 채널 믹스 방법론 | 매체별 도달·빈도 최적화 기준 |
| 카피 톤 방법론 | 브랜드·시즌별 톤앤매너 패턴 |

## 비민감 데이터 활용 시 고려할 점

- 타깃 데이터는 세그먼트·집계 형태로 가공
- 미발표 제품·가격 정책은 입력 금지
- 광고 사전 심의·승인 자료는 별도 절차 유지

## 검증 시 일반적으로 보는 항목

- 브랜드 톤·표현 가이드 준수
- 광고 표시·심의 규정 위반 여부
- KPI가 측정 가능한 수준으로 구체적인가
