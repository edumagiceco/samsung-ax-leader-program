# 50. 가격 전략 (pricing-strategy)
{: .fs-7 }

원가·경쟁가·가치 기반 가격을 시뮬레이션하는 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 원가 분석, 경쟁 가격 조사, 가치 기반 평가, 시나리오 시뮬레이션
- **출력 형태**: 가격 전략 보고서, 시나리오별 손익 비교
- **사용 빈도**: 신제품 출시·캠페인 기획 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 원가 분석가 | 직접비·간접비·기여 마진 계산 |
| 경쟁 분석가 | 경쟁사 가격·할인 정책 조사 |
| 가치 평가자 | 고객 인지 가치·지불 의사 추정 |
| 시뮬레이터 | 가격대별 매출·이익 시나리오 |
| reviewer | 가격 정책·법규 정합성 점검 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 원가·경쟁·가치 통합 시뮬레이션 |
| 탄력성 분석 방법론 | 가격-수요 곡선 추정 |
| 할인·번들 방법론 | 패키지·할부·렌탈 비교 |

## 비민감 데이터 활용 시 고려할 점

- 원가 구조는 비율 또는 등급으로 가공
- 경쟁사 가격은 공개 정보만 사용
- 미공개 캠페인 정책은 입력 금지

## 검증 시 일반적으로 보는 항목

- 시뮬레이션 가정과 결과가 논리적으로 연결되는가
- 본사 가격 정책 범위 안에 있는가
- 법적 가격 표시·할인 규정과 충돌이 없는가
