# 47. 전략 프레임워크 (strategy-framework)
{: .fs-7 }

OKR·BSC·SWOT 등 전략 프레임워크를 통합 분석·작성하는 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: OKR 설계, BSC 매핑, SWOT 분석, 비전·미션 선언문
- **출력 형태**: 전략 실행 로드맵, 통합 전략 문서
- **사용 빈도**: 분기·연간 전략 수립 주기

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| OKR 설계자 | Objectives·Key Results 구조 설계 |
| BSC 분석가 | Balanced Scorecard 4 perspective 매핑 |
| SWOT 전문가 | 내·외부 환경 분석 |
| 전략문서작성자 | 통합 전략 문서 작성 |
| reviewer | 프레임워크 간 일관성 교차 검증 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 4명의 분석 결과를 통합 문서로 연결 |
| OKR-BSC 매핑 방법론 | OKR과 BSC 간 정합성 점검 절차 |
| 시나리오 평가 방법론 | 전략 옵션 비교·우선순위 기준 |

## 비민감 데이터 활용 시 고려할 점

- 전략 수립 시 내부 실적·인사·재무 데이터는 익명화 또는 비율화 가능
- 외부 시장·경쟁사 데이터는 공개 자료 사용 권장
- 임원진 미공개 의사결정 사항은 입력 금지

## 검증 시 일반적으로 보는 항목

- 프레임워크 간 항목 누락·중복 여부
- OKR이 BSC 4 perspective에 모두 매핑되는가
- 비전·미션과 KR 간 정합성
