# 53. 재무 모델링 (financial-modeler)
{: .fs-7 }

수익 모델·비용 구조·시나리오 분석·밸류에이션을 통합 다루는 재무 모델 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 매출 모델, 비용 구조, 시나리오 분석, DCF·배수 밸류에이션
- **출력 형태**: 재무 모델 워크시트, 시나리오 비교표, 밸류에이션 보고서
- **사용 빈도**: 신사업 검토·투자 의사결정 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 수익 모델러 | 매출 추정·가격·물량 가정 |
| 비용 분석가 | 고정비·변동비·CapEx 구조 |
| 시나리오 플래너 | 낙관·기본·비관 시나리오 |
| 밸류에이션 전문가 | DCF·배수 평가 |
| reviewer | 가정 일관성·수식 정합성 점검 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 수익→비용→시나리오→밸류에이션 흐름 |
| 민감도 분석 방법론 | 핵심 변수의 영향 정량화 |
| DCF·배수 방법론 | 할인율·종가 가치 추정 기준 |

## 비민감 데이터 활용 시 고려할 점

- 미공개 사업 수치는 비율 또는 가정값으로 가공
- 동종 업계 배수는 공개 자료 사용
- 미발표 인수·합병 계획은 입력 금지

## 검증 시 일반적으로 보는 항목

- 가정값이 시나리오 간 일관되게 적용되는가
- 민감도 분석에서 결과가 합리적 범위에 머무는가
- 밸류에이션 결과가 동종 업계 범위 안에 있는가
