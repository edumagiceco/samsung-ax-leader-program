# 51. 투자자 보고서 (investor-report)
{: .fs-7 }

재무 실적·KPI·시장 동향·전략 업데이트를 통합한 투자자 보고서 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 분기·연간 실적, KPI 대시보드, 시장 동향, 전략 업데이트, 리스크 공시
- **출력 형태**: 투자자 보고서, 실적 발표 자료, IR 메시지
- **사용 빈도**: 분기 IR 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 재무 분석가 | 실적·재무비율·전년 대비 분석 |
| KPI 설계자 | 핵심 KPI 대시보드 구성 |
| 시장 분석가 | 산업·경쟁 동향 |
| 전략 작성자 | 향후 전략·가이던스 |
| reviewer | 공시 기준·표현 정합성 점검 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 실적→KPI→시장→전략 통합 흐름 |
| 공시 표현 방법론 | IR 메시지 톤·규제 표현 기준 |
| 비교 분석 방법론 | YoY·QoQ·peer 비교 패턴 |

## 비민감 데이터 활용 시 고려할 점

- 미공시 실적은 입력 금지 (Reg FD·공정공시 규정)
- 가이던스·전망은 승인된 범위 내 표현
- 동종 업계 비교는 공개 자료 한정

## 검증 시 일반적으로 보는 항목

- 수치 합산·전년 대비 계산이 정확한가
- 공시 의무 사항이 빠짐없이 포함되었는가
- 가이던스 표현이 규정에 부합하는가
