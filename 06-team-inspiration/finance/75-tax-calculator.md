# 75. 세금 계산·절세 전략 (tax-calculator)
{: .fs-7 }

소득·비용 분석부터 공제 항목 최적화·세액 계산·절세 시뮬레이션을 다루는 하네스 사례.
{: .fs-5 .fw-300 }

---

## 어떤 도메인의 하네스인가?

- **다루는 영역**: 소득·비용 분류, 공제 항목 최적화, 세액 계산, 절세 시뮬레이션
- **출력 형태**: 세액 계산 보고서, 신고 자료, 절세 옵션 비교
- **사용 빈도**: 분기·연간 신고 시점

## Agents 5명 구성 (예시)

| 에이전트 | 역할 |
|---|---|
| 소득 분석가 | 소득 구분·과세 표준 산정 |
| 공제 전문가 | 공제·감면 항목 최적화 |
| 세액 계산 엔진 | 세율·세액 계산 |
| 절세 전략가 | 합법 절세 옵션 시뮬레이션 |
| reviewer | 세법 정합성·계산 정확성 검증 |

## Skills 3개 구성 (예시)

| 스킬 | 역할 |
|---|---|
| 오케스트레이터 | 소득→공제→세액→시뮬레이션 흐름 |
| 세법 매핑 방법론 | 최신 세법·고시 반영 절차 |
| 신고 자료 방법론 | 신고서 양식·증빙 정리 |

## 비민감 데이터 활용 시 고려할 점

- 개인·법인 식별 정보는 코드화
- 미공개 거래·예정 거래는 입력 금지
- 세무사·회계사 자문 자료는 별도 보관

## 검증 시 일반적으로 보는 항목

- 최신 세법·고시가 반영되었는가
- 공제 한도가 정확히 적용되는가
- 절세 옵션이 합법 범위 안에 있는가
